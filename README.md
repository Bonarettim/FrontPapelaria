# 🛒 Frontend: Sistema de Gestão de Papelaria

Interface desenvolvida para consumir a API de gestão de vendas e comissões, utilizando tecnologias modernas e componentes de interface robustos.

## 🚀 Tecnologias
* **Framework:** React 19
* **Linguagem:** TypeScript
* **Estilização:** Bootstrap 5 (com Bootstrap Icons)
* **Gerenciamento de Estado:** React Native (Context API / Hooks)
* **Roteamento:** React Router DOM 7
* **Build Tool:** Vite

## 🛠️ Critérios de Avaliação
* **Componentização:** Estrutura organizada de componentes para reutilização e manutenção.
* **Consumo de API:** Chamadas estruturadas para comunicação com o backend Django.
* **TypeScript:** Tipagem estrita garantindo segurança e escalabilidade no desenvolvimento das interfaces.
* **Navegação:** Roteamento eficiente via React Router.

## ⚙️ Como rodar o projeto
1. Clone o repositório: `git clone https://github.com/Bonarettim/FrontPapelaria.git`
2. Instale as dependências: `npm install`
3. Inicie a aplicação: `npm run dev`
